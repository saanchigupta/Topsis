How It Works

TOPSIS stands for "Technique for Order of Preference by Similarity to Ideal Solution". It's a method for ranking alternatives based on multiple criteria.
File Format:
First column = option names
Other columns = evaluation criteria
All values must be numbers
Example Weights: 1, 2, 3
Means: first criterion is least important, third is most important
Example Impacts: +, -, +
Means: higher is better, lower is better, higher is better
Results:
Topsis Score: 0 to 1 (higher is better)
Rank: 1 = best option
